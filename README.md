# 📚 Library Management System

Simple library management system to practice Java OOP and PostgreSQL.

## 🎯 Learning Goals
- Java OOP (Classes, Inheritance, Composition)
- PostgreSQL integration with JDBC
- CRUD operations
- Data validation

## 🛠️ Tech Stack
- Java 11+
- PostgreSQL 15
- JDBC

## ✅ Progress - Week 1

**Day 1 (14/03/2026):**

- ✅ Member class (validations)
- ✅ Library class (composition, ArrayList)
- ✅ Basic CRUD (add, list)
- ✅ Testing with Main
